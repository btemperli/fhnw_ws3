## Setup

### Compass / Sass

1. Im Terminal ins Hauptverzeichnis navigieren

```
$ cd /weleda.ch
```

2. Compass installieren

```
$ gem install compass
```

3. Compass watch laufen lassen, um das styles.css-File zu generieren.

```
$ compass watch .
```
